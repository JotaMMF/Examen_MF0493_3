# Sabores del Mundo – PWA Gastronómica

## Archivos del proyecto

| Archivo | Descripción |
|---|---|
| `index.html` | Estructura semántica completa |
| `styles.css` | Estilos con variables CSS, responsive y hover |
| `app.js` | Lógica JS: datos, render, modal, búsqueda, PWA |
| `sw.js` | Service Worker con caché offline-first |
| `manifest.json` | Manifest PWA con nombre, iconos y colores |
| `icons/` | Iconos 192×192 y 512×512 para instalación |
